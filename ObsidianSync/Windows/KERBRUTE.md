## KERBRUTE
`pip3 install kerbrute`

```
git clone https://github.com/TarlogicSecurity/kerbrute
cd kerbrute
pip install -r requirements.txt
```
install in /opt/

example usage:
`./kerbrute.py -users users_file.txt -passwords passwords_file.txt -domain contoso.com`
